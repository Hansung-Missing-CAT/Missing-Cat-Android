# Missing PET — 프론트엔드 작업명세서 (WBS)

> 분실 반려동물 AI 매칭 모바일 앱 | 팀: 세나고 | 기술 스택: React + Vite + TypeScript (PWA) + Capacitor

---

## UI 공통 정책

| 항목 | 정책 |
|------|------|
| 타겟 환경 | Android 모바일 앱 (PWA → Capacitor 래핑) |
| 메인 테마 색상 | 빨강 (`#D32F2F`) + 화이트 (`#FFFFFF`) |
| 메인 폰트 | 프리텐다드 (Pretendard) |
| 하단 네비게이션 | 홈 / 실종신고 / 제보 / 채팅 (4개 탭) |
| 이미지 비율 | 리스트 내 고양이 사진: 1:1 정방형 또는 4:3 |
| 박스 곡률 | 카드/썸네일: 12dp, 버튼: 8dp 둥근 모서리 |

---

## 기술 스택

| 구분 | 기술/도구 |
|------|-----------|
| 프론트엔드 | React 18 + TypeScript + Vite 5 |
| 상태 관리 | Zustand 또는 React Context API |
| 스타일링 | CSS Modules / Tailwind CSS (택 1) |
| 라우팅 | React Router v6 |
| PWA | vite-plugin-pwa, Workbox |
| 네이티브 래핑 | Capacitor (카메라, 푸시 알림) |
| 지도 API | 카카오맵 또는 네이버맵 JavaScript API |
| 폰트 | Pretendard |
| 빌드/배포 | Android Studio (APK/AAB 서명), Play Store |

## 폴더 구조

```
src/
├── components/          # 공통 UI 컴포넌트 (Button, Card, Modal, NavBar 등)
├── pages/
│   ├── splash/          # 스플래시 화면
│   ├── auth/            # 로그인, 회원가입
│   ├── home/            # 메인 홈, 피드, 상세, 검색, 알림
│   ├── report/          # 실종 신고 (반려인 플로우, Step 1~5)
│   ├── tipoff/          # 제보하기 (신고자 플로우, AI 매칭)
│   ├── chat/            # 채팅 목록, 채팅방
│   └── settings/        # 프로필, 설정, 내 게시글 관리
├── hooks/               # 커스텀 훅 (useAuth, useLocation 등)
├── services/            # API 호출 모듈
├── stores/              # 전역 상태 (Zustand/Context)
├── utils/               # 유틸 함수
├── assets/              # 이미지, 아이콘, 폰트
├── styles/              # 글로벌 CSS, 테마 변수
└── types/               # TypeScript 타입 정의
```

## 화면 흐름 요약

**1) 시작 및 로그인**
앱 실행 → 스플래시 화면 → 로그인/회원가입 화면 (이메일, 비밀번호) → 메인 홈 화면

**2) 제보 프로세스 (신고자)**
하단 제보 탭 → 사진 올리기 (3장 이상) + 발견 위치 입력 → AI 유사도 검증 → 유사 실종 사례 명단 → 상세 진입

**3) 실종 신고 프로세스 (반려인)**
하단 실종 신고 탭 → 위치 입력 → 사진 업로드 (5장 이상) → 반려동물 정보 → 실종 시기 → 사례금 → 등록 완료

**4) 매칭 및 커뮤니케이션**
제보/유사도 검증 → 알림 발생 → 알림창 확인 → 채팅 탭 → 1:1 채팅 → 사진/장소 공유

## 용어 정의

| 용어 | 의미 |
|------|------|
| 신고자 (Reporter) | 길에서 발견한 동물의 사진을 업로드하는 사용자 (= 사진 업로더) |
| 반려인 (Owner) | 분실한 반려동물의 실종 게시글을 작성하는 사용자 |
| 매칭 | AI 사진 유사도 비교 결과 |
| 피드 | 분실 반려동물 게시글 목록 (메인 홈) |

## AI 매칭 4단계 흐름

```
1. 신고자가 발견한 동물 사진 + 위치 정보 업로드
2. AI가 분실 반려동물 게시글과 사진 유사도 비교
3. 일치 가능성이 높은 후보를 반려인에게 알림
4. 반려인과 신고자가 1:1 메시징으로 확인
```
